# Alarm Guide

Cell: Lakeside Packaging Line 3
Asset: PX-22 palletizer wrist (PX-22)
Fault: alarm 44 overcurrent during startup after a gripper swap

## Approved Checks

- inspect the wrist harness for pinched or over-tensioned cabling
- run low-speed homing before attempting a full-speed cycle
- verify the correct counterweight profile for the new gripper is loaded

## Forbidden Shortcut

Do not disable torque limits to push through startup faults.
